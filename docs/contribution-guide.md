# Contribution Guide

## How to Engage
- Join the community Discord for discussions and early feedback.
- Check existing issues before filing a new bug or feature request.
- Use the issue templates under `.github/ISSUE_TEMPLATE/`.

## Contribution Principles
- Focus on improving human-AI collaboration and workflow clarity.
- Prefer changes that strengthen reusable agents, workflows, and documentation.

## Before Opening a PR
- Confirm the issue or feature request is tracked.
- Ensure changes align with BMad's methodology goals.
- Run linting and tests before submitting.
